# 事業計画書：高収益農業・意思決定支援ツール『APE (Agri-Profit Engine)』

## 1. コンセプト (Concept)

### 「負けない経営」を創る意思決定エンジン

**「作ってから売り先を探す」マッチングアプリではなく、「利益が出るものを特定してから作る」ための経営戦略ツール。**

多くの農業アプリが野菜が出来た**後**のマッチングに注力するのに対し、APEは野菜を作る**前**の意思決定こそが収益の9割を決めると定義します。マッチング機能はあくまで付加価値であり、本質は**「勝算のない戦（作付）を避ける」**ことにあります。

## 2. 解決する課題 (The Pain)

農業経営は「3つの不確実性（博打）」に支配されており、特に新規就農者にとって致命的なリスクとなっています。

### ① 「何を作る？」の博打（情報の非対称性）
多くの農家が「富良野なら人参」といった地域の慣習や、「昨年高かったから」という単純な動機で作付品種を決めてしまいます。しかし、個別の畑の土壌条件や、来年の市場トレンドとは乖離していることが多く、結果として**「土地に合わず育たない」「豊作貧乏で利益が出ない」**という失敗を繰り返しています。

### ② 「どこに売る？」の博打（ミスマッチ）
近隣の飲食店が「本当は何を欲しがっているか（例：実はイタリアンパセリを探している等）」を知る術がありません。需要が見えないため、結局は安価な市場出荷（JA）に依存せざるを得ず、手取りが増えません。

### ③ 「失敗できない」プレッシャー
体力のない小規模農家や新規就農者にとって、一度の作付ミスは廃業に直結します。大規模農家のように「数撃ちゃ当たる」戦略は取れず、確たる指標もないまま**「経験と勘」という名のギャンブル**を強いられています。

## 3. ソリューション (Solution)

APEは、「経験と勘」を「物理とデータ」に置き換え、確実に利益を出すためのソリューションを提供します。

### ① 土壌・気象データに基づく「真の適地適作」診断
その土地（畑単位）のデータを分析し、生物学的・経営的に最もパフォーマンスが良い作物を提案します。
単に「育つ」だけでなく、労働コストまで含めた**手残り利益（純利益）のシミュレーション**を行います。

> **なぜ正確に予測できるのか？（3つの科学的エンジン）**
> *   **🔬 Agro-Physics Engine (栽培物理エンジン):**
>     Wageningen大学の**Yield Gap理論**と**FAO-GAEZモデル**（日射量×光利用効率）を用い、過去の統計ではなく生物学的な「理論上の最大収量」を算出します。
> *   **📊 Bio-Economic Engine (経営経済エンジン):**
>     **農研機構**の標準労働データを実装し、単なる売上ではなく「時給換算でいくら残るか」を算出。**SERFモデル**により、リスク（倒産確率）も加味した「負けない作物」を選定します。
> *   **🧠 Behavioral Nudge Engine (行動変容エンジン):**
>     **MITのナッジ理論**を応用し、「わかっているけど変えられない」農家の背中を押し、新しい作物への転換（行動変容）を促します。

### ② 地域需要の可視化機能「ローカル・デマンド・ビュー」
半径数km圏内の飲食店や実需者が「今、どの野菜を、どれくらい求めているか」を地図やリストで見える化します。
不確実な「遠くの市場」ではなく、**「隣のレストランが欲しがっている（＝売れることが確定している）野菜」**を作ることで、物流費を抑えつつ確実に売り切る経営が可能になります。

## 4. 競合優位性 (Differentiation)

既存のアグリテックが「作った後の管理」や「販売」に集中しているのに対し、APEは**「作る前の意思決定」**に特化しています。

| | 既存の農業アプリ (管理・マッチング) | **APE (本ツール)** |
| :--- | :--- | :--- |
| **タイミング** | 野菜ができた**「後」** | 野菜を作る**「前」** |
| **アクション** | 懸命に売り先を探す・記録をつける | **勝算（利益）を確認してから作る** |
| **リスク** | 売れ残るリスク、安値のリスク | 高利益・高回転が見込める状態で作付開始 |
| **技術の透明性** | ブラックボックス（AI任せ） | **ホワイトボックス**（物理モデルと統計に基づく説明可能性） |
| **提供価値** | マッチング・効率化 | **経営戦略（生存率の向上）** |

## 5. ターゲット (Target)

**失敗のリスクを最小限に抑えたい「チャレンジャー」たち**

*   **新規就農者:**
    「この土地で何を作れば食べていけるのか？」という最初の問いに対して、データに基づく正解を求める層。
*   **小規模農家:**
    一発逆転ではなく、着実に利益を積み上げ、持続可能な経営を目指す層。
*   **法人就農者:**
    リソースの最適化を図り、すでに育てている作物と作業時期が被らない「第2、第3の柱（高収益作物）」を探している層。

## 6. ビジネスモデル (Model)

*   **SaaS 利用料 (ToB):**
    詳細な土壌診断と収益シミュレーション機能の提供（月額2,980円〜）。
*   **自治体導入プラン (ToG):**
    地域の「耕作放棄地」のポテンシャル可視化や、新規就農者誘致のための強力なツールとして自治体に提供（年額ライセンス）。
*   **マッチング手数料:**
    「ローカル・デマンド・ビュー」を通じた地域内流通の成約手数料（将来構想）。

## 7. 創業者の原体験 (Founder's Story)

**「儲かる作物は固定されていない」**

私はトマト農家の家に生まれ、自身も農業経営を行う中で、多くの失敗を経験しました。椎茸、グリーンピース、イチゴ、ネギ……。情熱を持って作っても、相場やコスト構造の変化で撤退を余儀なくされる。「良いものを作る」だけでは生き残れない現実を痛感しました。

その後、青果市場に勤務し全体を俯瞰した時、さらに根深い問題に気づきました。
「昔はこの野菜で儲かった」という成功体験や、「立派な選果場を作ったから」という設備へのサンクコストに縛られ、今の気候や市場に合わなくなった作物を作り続けて疲弊する農家がいかに多いか。

農業はもっと自由で、科学的であるべきです。
土地の声（土壌データ）と市場の声（需要データ）を聞けば、その場所で「今、一番輝く作物」が見えてくる。
APEは、過去のしがらみではなく、未来のデータに基づいて、農家が自信を持って経営判断を下すためのエンジンです。
